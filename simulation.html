Interactive Spatial Process Simulation

Spatial Simulation

This is an adaption of a conditional spatial simulation that I originally wrote in Python and adpated somewhat faithfully to JavaScript. This categorical simulation is constructed using process convolutions to blend a latent spatial process with Gaussian kernel functions. A process convolution is calculated for each category over the entire spatial domain. At each knot location used in the process convolution calculations, the process with the highest value is selected. This process is then used to classify the knot as that category.

A similar workflow is used to simulate a continuous spatial process which is conditioned on the categorical process. The repository for that project can be found here: Conditional Simulation. This is part of a larger project that investigates the effectiveness of Bayesian Process Convolution models to model spatial processes in sparse data spatial domains.
